# Чек-лист проверки безопасности рабочего места

## Операционная система
- [ ] Установлена поддерживаемая версия ОС (не EOL).
- [ ] Включено автоматическое обновление системы.
- [ ] Установлены последние патчи безопасности.

## Учетные записи
- [ ] Используется сложный пароль (минимум 12 символов).
- [ ] Гостевая учетная запись отключена.
- [ ] Настроена блокировка экрана (тайм-аут 5-15 мин).

## Шифрование и защита
- [ ] Включено шифрование диска (BitLocker / FileVault / LUKS).
- [ ] Включен межсетевой экран (Firewall).
- [ ] Установлен и активен антивирус / EDR.

## Сеть
- [ ] Домашний роутер защищён сложным паролем (не admin/admin).
- [ ] Прошивка роутера обновлена.
- [ ] Wi-Fi сеть использует шифрование WPA2/WPA3.

## ПО
- [ ] Удалено неиспользуемое ПО.
- [ ] Браузер обновлён до последней версии.
