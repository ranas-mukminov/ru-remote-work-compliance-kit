# ru-remote-work-compliance-kit

## English

`ru-remote-work-compliance-kit` is an open-source toolkit designed to help Russian companies and their remote employees establish secure remote work practices. It includes VPN configurations, policy templates, endpoint security check scripts, and an AI-powered helper to generate draft policies based on organizational profiles.

This project aims to address technical and organizational measures required by regulations like 152-FZ (Personal Data) and general security best practices (VPN, MFA, Disk Encryption).

**Disclaimer:** This is a technical toolkit, not legal advice. Always consult with your legal team.
