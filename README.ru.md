# ru-remote-work-compliance-kit

**Набор инструментов для безопасной удалённой работы (152-ФЗ, КИИ, Best Practices)**

## Что это

`ru-remote-work-compliance-kit` — это открытый репозиторий, содержащий шаблоны и скрипты для организации безопасной удалённой работы в российских компаниях.

Включает в себя:
- **VPN-конфиги** (WireGuard, OpenVPN) для сценариев split-tunnel и forced-tunnel.
- **Шаблоны политик** (удалённая работа, BYOD, обработка ПДн) в формате Markdown/ODT.
- **Скрипты проверки рабочих станций** (Linux, Windows, macOS) на соответствие требованиям ИБ (шифрование, обновления, антивирус).
- **AI-помощник** для генерации черновиков политик и оценки рисков на основе профиля организации.

## Для кого

- **SMB (Малый и средний бизнес):** у которых нет большого штата безопасников, но есть потребность в защите.
- **Интеграторы:** для быстрого развёртывания типовых решений у клиентов.
- **Internal DevSecOps:** для автоматизации проверок и стандартизации настроек.

## Быстрый старт

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/run-as-daemon/ru-remote-work-compliance-kit.git
cd ru-remote-work-compliance-kit

# Установка зависимостей (потребуется Python 3.10+)
pip install .
```

### Использование CLI

1. **Проверка рабочей станции:**
   ```bash
   ru-remote-kit assess-endpoint --format json --output report.json
   ```

2. **Генерация политики удалённой работы:**
   ```bash
   ru-remote-kit generate-policy --profile examples/org-profiles/smb-it-outsourcing.yaml --out policy.md
   ```

3. **Оценка рисков:**
   ```bash
   ru-remote-kit generate-risk-list --profile examples/org-profiles/fintech-remote-first.yaml
   ```
